# Crypto Dashboard (готово для GitHub Pages)

Минималистичный крипто‑дашборд на React + Vite с реальными данными CoinGecko и индексом страха/жадности.

## Как запустить локально

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173` (по умолчанию).

## Как собрать и выложить на GitHub Pages

1. Соберите проект:

```bash
npm run build
```

В папке `dist` появится готовый статический билд.

2. Создайте новый репозиторий на GitHub и залейте туда весь проект **или** только содержимое `dist`.

3. В настройках репозитория GitHub откройте:

`Settings` → `Pages` → **Source** → выберите:
- `Deploy from a branch`
- ветка `main` (или `master`)
- папка `/dist` (если заливаете весь проект) или `/` (если репозиторий состоит только из файлов билда).

4. Сохраните настройки. Через пару минут сайт будет доступен по адресу, который покажет GitHub Pages.

> Если репозиторий называется, например, `crypto-dashboard`, и вы деплоите **весь проект**, убедитесь, что `base: './'` в `vite.config.ts` — это уже настроено.
